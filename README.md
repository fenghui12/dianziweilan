# 电子围栏监控系统

这是一个基于YOLOv11的电子围栏监控系统，用于监控连廊区域，当人员接近危险区域时发出警告。

## 功能特点

- 实时人物检测
- 自定义警戒线设置
- 越界警告提示
- 简单易用的界面

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行主程序：

```bash
python src/main.py
```

2. 配置参数：
   - 在`config.yaml`中设置警戒线位置
   - 调整检测灵敏度和其他参数

## 最小可行产品（MVP）

当前版本为MVP，实现了基本的人物检测和警戒线越界提醒功能。
